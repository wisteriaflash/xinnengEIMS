/////////////////////////////////////////////
 DedeEIMS V1.1 系统简介
/////////////////////////////////////////////
Dede Enterprise Information Management System (简称DedeEIMS)是织梦科技为中
小型企业专门开发的一套建站系统，产品继承了织梦内容管理系统简单、易用的特点
，并且加入了如：多国语言、在线客服、招聘、产品订单等功能。
程序的内容创建获取及管理、安全灵活的程序架构，简单易懂的模板标签机制，方便
快捷的频道管理以及无限极拓展分类的需求，标准静态化更适合搜索引擎优化（SEO）
，让你能够轻松打造出专业的企业网站，帮你在互联网拓展公司业务，提升企业的核
心竞争力。

/////////////////////////////////////////////
 DedeEIMS 系统软件的环境需求
/////////////////////////////////////////////
1、Windows 平台：
IIS/Apache + PHP4/PHP5 + MySQL4/5
如果在windows环境中使用，建议用DedeEIMS提供的DedeAMPZ套件以达到最佳使用性能。

2、Linux/Unix 平台
Apache + PHP4/PHP5 + MySQL3/4/5 (PHP必须在非安全模式下运行)

建议使用平台：Linux + Apache2.2 + PHP5.2 + MySQL5.0

3、PHP必须环境或启用的系统函数：
allow_url_fopen
GD扩展库
MySQL扩展库
系统函数 ―― phpinfo、dir

/////////////////////////////////////////////
 DedeEIMS 系统软件的安装说明
/////////////////////////////////////////////
1. 上传 upload 目录中的文件到服务器，我们建议使用网站根目录
2. 设置目录属性（windows 服务器可忽略这一步）
   详细可以查看DedeEIMS的帮助中心说明
3. 执行安装脚本
   请在浏览器中运行 install 程序，即访问 http://您的域名/install/ 
4. 参照页面提示，进行安装，直至安装完毕
